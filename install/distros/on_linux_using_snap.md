To easily install Crystal on a Linux distribution you can use [Snap](https://snapcraft.io/crystal). See the [blog post](https://crystal-lang.org/2019/06/19/snapcraft-summit-montreal.html)Â for additional information.

```
sudo snap install crystal --classic
```

You can also install the latest nightly build by using the `edge` channel.
```
sudo snap install crystal --classic --edge
```
